# MESC (Multimodal Emotional Support Conversation)

## Overview
MESC is a **multimodal emotional support conversation** dataset for mental health–oriented dialogue research. It provides emotional support annotations and multimodal cues across **text, audio, and video**.

## Official Source (Data & Code)
- GitHub repository: https://github.com/chuyq/MESC

## Data Origin
The dataset is sourced from the TV show **In Treatment** (Seasons 1–3), which depicts psychotherapy sessions and related consultations.

## What’s Included (in the official repo)
- Ready-to-use splits: `train.*`, `val.*`, `test.*` (text splits with extracted multimodal emotion cues provided in the release)
- `MESC.csv`: contains timestamps for segments; can be used to download/process raw videos and extract multimodal features by yourself
- Additional repo materials (e.g., strategy definitions, video examples)
